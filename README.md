# Billing_Software

GST Billing & Inventory Management System
A complete Billing Software built using Spring Boot, Thymeleaf, and MySQL, designed for businesses to manage invoices, purchases, inventory, GST calculation, and reporting in a clean and responsive dashboard.
🚀 Features

•	• Customer Management (Add, Edit, Delete, List)
•	• Product Management with Stock Control
•	• Invoice Generation with GST Calculation
•	• Purchase Management with Supplier Tracking
•	• Dashboard with Sales, Net Revenue, and GST Charts (Chart.js)
•	• Expense Management with Category Filter
•	• Export to PDF & Excel (Invoices, Purchases, Expenses)
•	• Top 5 Products & Customers by Sales
•	• Daily & Monthly Sales/Purchase Reports
•	• Clean UI using Thymeleaf & Custom CSS (No Bootstrap)
🛠️ Tech Stack
•	• Backend: Java, Spring Boot, Spring Data JPA
•	• Frontend: Thymeleaf, HTML/CSS, JavaScript, Chart.js
•	• Database: MySQL
•	• PDF Export: iText (Lowagie)
•	• Excel Export: Apache POI
🧪 Setup & Run
Prerequisites:
- Java 17+
- Maven
- MySQL Server
📁 Modules
•	• CustomerModule - CRUD operations for customers
•	• ProductModule - Manage products and track stock
•	• InvoiceModule - Generate and export GST invoices
•	• PurchaseModule - Track purchases and supplier details
•	• ExpenseModule - Record and categorize expenses
•	• Dashboard - Graphs, top customers/products, analytics
•	• Reports - Sales/Purchase/Customer history reports
